## RxJavaDemo
前言：本项目使用到了Retrofit2 + Dagger + RxJava。
- Retrofit2：Restful 接口网络框架，它封装的okHttp，让网络请求更简洁。
- Dagger：是依赖注入框架，根据注解自动生成代码（基于Android-Apt），使得项目更加整洁，和MVP 架构配合使用。
- RxJava：更加广泛的观察者模式，用于解决异步请求，和Retrofit 可以配合使用。
